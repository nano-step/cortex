What Cortex Does

  • Search YOUR code → match YOUR style
  • 3-tier memory across sessions
  • Full repo indexing + dependency graphs
  • 25+ tools: git, vision, code advisor, web

The 6-Layer Cortex Brain

1

Understanding — Brain Engine

AST parsing → Voyage AI embeddings (1024-dim) → SQLite FTS + Qdrant vectors → Agentic RAG with decompose/search/boost/verify pipeline. Not keyword search — semantic understanding of your code.

2

Thinking — Pipeline Engine

Every query goes through: Hooks → Intent Classifier → Path Router → Agent/Tool/RAG → Verify. Five execution paths: orchestrate (multi-agent), skill_chain (ReAct), standard (RAG), perplexity (web), slash_command (direct).

3

Acting — 25+ Tools

Code Advisor (TabNine-inspired pattern matching), Vision (FREE image analysis), Artist (image generation), Git Analysis (contributors, history), Config Search, Web Search — all called automatically by the LLM.

4

Remembering — 3-Tier Memory

Core Memory (always in context: your style). Archival Memory (past decisions, semantic search). Recall Memory (conversation history). Cortex remembers what you prefer and how you code.

5

Collaborating — 12 Agents

Sisyphus (ultraworker), Hephaestus (deep debugger), Prometheus (planner), Atlas (heavy lifter), Oracle (consultant), Explore, Librarian + 5 specialists. Each has its own model, strategy, and personality.

6

Learning — Self-Improvement

Tracks your accepts, rejects, follow-ups, copies. Learned Reranker adjusts search weights. Feedback loop: the more you use Cortex, the better it understands what YOU find useful.
